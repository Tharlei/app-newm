NewMTest make with React Native

## Installation

First run npm install

```sh
npm install
```

## Server Init

_Listen on device or emulator_

```sh
npm start
```

Connect device android in computer and look in terminal

```sh
adb devices
```

When everything is right, type in the terminal

```sh
npm run android-dev
```
